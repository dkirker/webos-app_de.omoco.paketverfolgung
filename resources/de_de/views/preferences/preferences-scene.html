Einstellungen
Ansicht
Hier kann die automatische Sortierung der Liste festgelegt werden. Die Liste wird immer dann automatisch sortiert, wenn die App gestartet wird, eine Sendung hinzugefügt wird, oder die Liste aktualisiert wird.
Legt fest welches Datum in der Liste der Sendungen angezeigt werden soll. Es kann zwischen dem Erstellungsdatum und der letzten Statusänderung ausgewählt werden.
   Liste einfärben:
Wenn aktiviert, werden die Pakete in der Liste je nach Unternehmen farbig hervorgehoben.
   Details einfärben:
Wenn aktiviert, wird auch die Detailansicht je nach Unternehmen eingefärbt.
   Karte anzeigen:
Den Standort der Sendung in der Detailansicht auf einer Karte zeigen.
Zoomeinstellung für die große Kartenansicht. 2 ist weit entfernt, 10 ist max. rangezoomt.
   Cache benutzen:
Wird der Cache benutzt, werden in der Detailansicht schon die letzten bekannten Details angezeigt. Sobald die neuen Informationen geladen sind, werden diese dann angezeigt.
   Dunkles Design:
Schaltet das alternative, dunkle Palmdesign an. Es ist nicht komplett angepasst, bietet aber eine Alternative zum Standarddesign.
Aktualisierung
   Aktual. bei Start:
Wenn aktiviert, wird bei jedem Start und beim Aufruf vom Dashboard aus die komplette Liste der Sendungen aktualisiert.
   Aktual. bei Neu:
Legt fest ob eine Sendung nach dem Hinzufügen direkt aktualisiert werden soll.
   Automatisch Aktual.:
Die App kann im Hintergrund die Liste automatisch aktualisieren. Dazu muss aber die App und/oder das Dashboard laufen.
   Nur 6-20 Uhr Aktual.:
Es wird nur in dem Zeitraum von 6-20 Uhr aktualisiert. Die Aktualisierung beginnt morgens dann wieder zwischen 6 und 7 Uhr.
   Auch kleine Änder.:
Normalerweise wird nur bei den fünf großen Statusänderungen der Status gespeichert und der Benutzer benachrichtigt. Wenn diese Option aktiviert ist, wird jede Detailänderung berücksichtigt.
Benachrichtigung
   Benachr. anzeigen:
Wenn aktiviert, wird bei einer Änderung eine Benachrichtigung eingeblendet. Zusätzlich kann ein Klingelton abgespielt werden.
Klingelton
   Dashboard anz.:
Das Dashboard ist der Bereich unten, in dem sich laufende Programme als kleine Icons sammeln. Es kann dazu dienen, dass die automatische Aktualisierung aktiv bleibt. Außerdem werden Statusänderungen dort gesammelt angezeigt. Die App kann direkt aus dem Dashboard gestartet werden.
Import / Export
  Push aktivieren:
Bookmarklet zusenden
Wenn Push aktiviert ist, können Sendungen direkt vom Computer aus zum Handy geschickt werden. Bei neuen Sendungen wird dann die Information direkt übernommen. Damit das funktioniert muss einmalig ein Bookmarket eingerichtet werden. Eine genauere Anleitung erscheint beim Einrichten des Bookmarklets.
Sendungen importieren
Eine Liste mit Sendungen im CSV-Format kann hier importiert werden. Die Datei kann auf der Medienpartition oder im Web liegen. Alternativ kann auch aus der Zwischenablage importiert werden.
Als E-Mail exportieren
Export der aktuellen Sendungen als E-Mail, zum späteren Import geeignet. Bei dieser Art des Exports ist die Anzahl der Sendungen nicht limitiert.
Auf Gerät exportieren
Export der aktuellen Sendungen als Textdatei. Die Datei wird direkt auf dem Gerät gespeichert. Hinweis: Dies ist nur über den Umweg eines externen Servers möglich. Die Sendungsdaten werden dort nicht gespeichert. Bei dieser Art des Exports werden maximal ca. 100 Sendungen unterstützt.
Sonstiges
   Updates suchen:
Die App kann beim Start automatisch nach Updates und Störungsmeldungen suchen. Darüber kann der Entwickler sie informieren, falls es Neuerungen oder Probleme mit der Aktualisierung von Sendungen gibt.
   Beta Unternehmen:
Hiermit werden Unternehmen freigeschaltet die noch nicht vollständig getestet worden sind. Ich freue mich weitere Sendungs-nummern und Probleme per Mail zu bekommen, damit die Dienste weiter getestet werden können. Damit die Einstellung aktiv wird, muss die App neu gestartet werden.
