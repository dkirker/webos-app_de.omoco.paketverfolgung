Preferences
View
Set the order of the packages list. The list will be ordered automatically when starting the app, adding a new package or when refreshing it.
Set the shown date of the packages in the list. Choose between the creation date or the time of the last update.
   Colorize list:
If activated the packages will be highlighted in the color corresponding the each package service.
   Colorize details:
This will highlight the details view.
   Show Map:
Show package location in a map in details view.
Zoom level for the map after clicked on it. 2 is zoomed out, 10 ist max zoomed in.
   Activate cache:
When using the cache the package information will be saved and loaded from local storage before starting the web request. When there is updated information available the cached information will be replaced.
   Dark theme:
Use the alternative dark webOS theme. It is not 100% supported but definitely worth a try for people who love it darker.
Refreshing
   Refresh on start:
Refresh the list of packages when starting the app from the launcher or from the dashboard.
   Refresh on add:
Refresh a package after adding it to the list.
   Auto-Refresh:
The app is able to refresh the package data in background. To use this feature the app and/or the dashboard has to be running. If you close the the app completely (card and dashboard) it will be stopped. So if you like to have updates leave the dashboard running.
   Only 6am - 8pm:
To save power refreshing can be disabled over night. It will start on the next day automatically between 6am and 7am.
   Detailed notifications:
Normally a new status is saved only when there is one of the five big updates. With this setting on you will get notified on every small status change of a package.
Notifications
   Show notifications:
Activate notifications. Additionally an alarm sound can be set.
Ringtone
   Show dashboard:
The dashboard is the area on the bottom of the screen where currently running apps can display small icons. When the dashboard icon of this app is visible the app will keep running and background notifications are available. Every update of a package will be displayed. The main app can be started directly from the dashboard.
Import / Export
  Activate push:
Send bookmarklet
With the Push service you can send a tracking number directly from your computer to the app. Before using it you have to install a bookmarklet. More information about this when you hit the button.
Import Packages
Import packages from a .csv file. The file can be located in the media partition or downloaded directly from the web. You also can import packages from the clipboard.
Export via email
Export all of your packages via email. You can use this to backup your list and import it later. The number of packages is not limited.
Export to device
Export all of your packages directly to the device. Notice: To make this possible the list is sent to a server and downloaded from there. No information will be saved there. The maximum number of packages to export is 100.
Miscellaneous
   Check for updates:
Get the latest updates about this app.
   Beta services:
Unlock Services which are in beta state right now. There can be bugs or problems with these services. Help to improve them by sending me tracking numbers via email. To activate this setting the app must be restarted.
